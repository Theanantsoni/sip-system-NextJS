# 💰 SIP Planner - Next.js Web App

A modern and responsive **financial planning web application** built using **Next.js**, designed to help users calculate SIP returns, explore mutual funds, and make smarter investment decisions with a clean UI and intelligent suggestions.

---

## 🚀 Features

- ⚡ Built with Next.js (App Router)
- 🔐 User Authentication (Login & Register)
- 📊 SIP Calculator with real-time results
- 📈 Investment growth visualization (charts)
- 🤖 Smart mutual fund suggestions (AI-based logic)
- 💰 Risk-based fund filtering (Low / Mid / High)
- 📚 Beginner-friendly financial articles
- 📄 Multiple pages (Home, SIP, Mutual Funds, Articles, Contact)
- 🎨 Clean, modern, and responsive UI
- ⚡ Optimized performance and fast rendering

---

## 🧱 Tech Stack

- Next.js (App Router)
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- Next.js API Routes (Backend logic)
