# 📊 SIP Planner - Next.js Web App

**Author:** Anant Soni

A modern full-stack investment planning web application built using **Next.js**, designed to help users calculate SIP returns, understand investment concepts, and get smart mutual fund suggestions.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 📈 SIP Calculator with real-time calculation
- 📊 Investment growth visualization (charts)
- 🤖 AI-based SIP & Mutual Fund Suggestions
- 📂 Dynamic data handling using APIs
- 🎯 Risk-based recommendation system
- 📚 Beginner education (SIP concepts & guides)
- 🎨 Fully responsive modern UI
- ⚡ Fast performance with Next.js App Router

---

## 🧱 Tech Stack

- Next.js (App Router)
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- API Routes (Next.js backend)

---

## 📁 Project Structure

```bash
src/
├── app/
│   ├── page.tsx              # Home Page
│   ├── sip/                 # SIP Calculator
│   ├── mutualFunds/         # Fund Suggestions
│   ├── articles/            # Education Section
│   ├── contact/             # Contact Page
│   ├── login/               # Login Page
│   ├── register/            # Register Page
│   ├── api/                 # Backend APIs
│
├── components/
│   ├── HomePage.js
│   ├── LoginForm.js
│   ├── SipForm.js
│   ├── SipResult.js
│   ├── MutualFunds.js
│
├── utils/
│   ├── sipCalculator.js     # Calculation logic
│   ├── sipApi.js            # API calls


🖥️ Installation & Setup
# Clone repository
git clone https://github.com/Theanantsoni/TextUtils-ReactJS

# Go to project folder
cd sip-system

# Install dependencies
npm install

# Run development server
npm run dev

👉 Open in browser:
http://localhost:3000
```
